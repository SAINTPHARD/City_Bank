# city_bank interface

Interface web em React para o projeto `city_bank`, criada com Vite.

## Funcionalidades

- Dashboard com saldo total, contas e acoes rapidas.
- Formulario visual para pagamento de boleto.
- Lista de servicos bancarios.
- Transacoes recentes com entradas e saidas.
- Area de cartoes com limite utilizado.
- Painel de investimentos com resumo de rentabilidade.
- Layout responsivo para desktop, tablet e celular.

## Como executar

Instale o Node.js e depois rode:

```bash
npm install
npm run dev
```

Para gerar a versao de producao:

```bash
npm run build
```
